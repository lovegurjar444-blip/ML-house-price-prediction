
# House Price Prediction using Linear Regression

## Project Overview
This project builds a Machine Learning model to predict house prices using Linear Regression. 
The model uses housing data such as area and number of bedrooms to estimate the price of a house.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
The dataset contains information about houses including:
- Area of the house
- Number of bedrooms
- Price of the house

## Steps in the Project
1. Import required libraries
2. Load the dataset
3. Select features and target variable
4. Train the Linear Regression model
5. Make predictions
6. Evaluate model performance

## How to Run the Project

Clone the repository:

git clone https://github.com/your-username/house-price-prediction.git

Go to the project folder:

cd house-price-prediction

Install required libraries:

pip install -r requirements.txt

Run the model:

python train_model.py
